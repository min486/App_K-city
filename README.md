# 🏙️ K-city

Android 개인 프로젝트

## 프로젝트 소개

관심있는 한국의 도시에 대한 정보를 제공하는 앱

### 🛠️ 사용 기술

- Jetpack Compose

- MVVM
- Navigation - compose
- Splash Screen

### 🖥️ 개발 환경

- Programming Language : `Kotlin`
- IDE : `Android Studio`
- Minimum SDK Version : `28`
- Compile SDK Version : `33`
- Target SDK Version : `33`
